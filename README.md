## ✨ Features

- **Futuristic Design** — Dark grid background, scan-line animation, glowing cyan accents
- **Terminal Hero Card** — Animated JSON-style profile card with blinking cursor
- **Scroll Animations** — Intersection Observer–powered fade-up on every section
- **Hover Interactions** — Lift effects, glowing borders, animated underlines
- **Zero Dependencies** — Pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step
- **Fully Responsive** — Works cleanly on mobile, tablet, and desktop
- **Fast Load** — Single HTML file, no external JS bundles, Google Fonts only

## 🚀 Getting Started

### View locally

Just open the file in your browser:

```bash
# Clone the repo
git clone https://github.com/Nxtsubham/portfolio.git

# Open in browser (macOS)
open index.html

# Open in browser (Linux)
xdg-open index.html

# Open in browser (Windows)
start index.html
```

No `npm install`. No build process. No dependencies.

## 🎨 Customization

All content lives in the HTML file. To update:

| What to change | Where to look |
|---|---|
| Name, role, bio | `<!-- HERO -->` section |
| Projects | `<!-- PROJECTS -->` section |
| Skills & tags | `<!-- SKILLS -->` section |
| Experience | `<!-- EXPERIENCE -->` section |
| Contact links | `<!-- CONTACT -->` section |
| Color palette | `:root` CSS variables at the top |

### Color variables

```css
:root {
  --accent:  #00E5FF;   /* Cyan — primary highlight */
  --accent2: #7B61FF;   /* Purple — secondary */
  --accent3: #FF6B6B;   /* Red — badges */
  --bg:      #050A0F;   /* Deep dark background */
  --text:    #E8F4F8;   /* Primary text */
  --muted:   #6B8A9A;   /* Subdued text */
}
```
